# GenshinWishCalculator


## 功能介绍
>本插件可以根据玩家当前拥有纠缠之缘数目和角色池、武器池的水位、保底情况，计算出抽取指定数目五星up角色和特定的五星up武器的概率

>使用本插件需要python环境和numpy库，如果觉得不方便，你还可以使用[纯js版本](https://github.com/MSIsunny/GenshinWishCalculator),但是运行的速度将会大大降低
